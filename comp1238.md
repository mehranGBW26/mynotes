## Week 1 - History
- Mesopotamia is believed to be the earliest writing we know of
- Cunieform stands for wedge shaped in Latin
- Printing is almost as old as writing
  
## Week 2 - Bits and Bytes
- Volatile is for short-term memory & Non-Volatile is for long-term memory
- Ascii (American Standard Code for Information Interchange) was developed for early communication
- Signed can be positive or negative & unsigned is only non-negative
  
## Week 3 - Text Structure
- HTML (Hypertext Markup Language) is the most common markup language
- Non-alphanumeric is all the other printable characters
- Alphanumeric: Letters (Alpha) and numbers (numeric)
  
## Week 4 - Keyboards and Editors
- QWERTY is the most common keyboard layout globally
- Touch Typing is when you can type without looking at the keys
- Modifier Keys only modify other keys
  
## Week 5 - Command Line Interfaces (CLI)
- SSH (Secure Shell) is for connecting to remote computers over a network
- Back then CLI was the only way to interact with computers
- CLI might be faster for certain work
